# 配件

## 主板

主要厂商是Intel(英特尔)和AMD。

### 厂商命名含义：

> B760
>
> B+7+60
>
> 前缀+系列+数字大就是好

### Intel前缀含义：

| **前缀** | **定位**            | **主要特性**                                        | **适用 CPU**                  | **典型型号** |
| -------- | ------------------- | --------------------------------------------------- | ----------------------------- | ------------ |
| X        | 旗舰/发烧级（HEDT） | 支持多核 CPU（如 i9-X 系列）、PCIe 通道多、超频支持 | Core X 系列（如 i9-10980XE）  | X299, X399   |
| Z        | 高端（可超频）      | 支持 CPU & 内存超频、多 PCIe 通道、高端扩展         | K/KF 后缀 CPU（如 i7-14700K） | Z790, Z690   |
| B        | 主流级              | 不支持 CPU 超频（但 B660+ 支持内存超频），性价比高  | 非 K 系列（如 i5-14600）      | B760, B660   |
| H        | 入门级              | 基础功能，不支持超频，扩展性较低                    | i3/奔腾/赛扬                  | H610, H770   |

### AMD后缀含义：

| **前缀** | **定位**  | **主要特性**                       | **适用 CPU**             | **典型型号** |
| -------- | --------- | ---------------------------------- | ------------------------ | ------------ |
| X        | 旗舰/高端 | 支持超频、PCIe 通道多、扩展性强    | Ryzen 9/7（如 7950X）    | X670E, X570  |
| B        | 主流级    | 支持超频（AMD 全系支持），性价比高 | Ryzen 7/5（如 7600X）    | B650, B550   |
| A        | 入门级    | 基础功能，不支持 PCIe 4.0/超频     | Ryzen 3/速龙（如 5600G） | A520, A620   |

### 如何区分 Intel 和 AMD 主板？

1. 看前缀：
   - Intel：Z、B、H、X（如 Z790、B760）。
   - AMD：X、B、A（如 X670、B650）。
2. 看数字规律：
   - Intel B 系列：第二位通常是 双数（如 B660、B760）。
   - AMD B 系列：第二位通常是 单数（如 B550、B650）。

### 推荐：

- AMD B650M
- Intel B760M

>
> RTX+50+60+Ti
>
> 系列+第几代+数字大就是好+有后缀就是好

- RTX5060
- RX6750GRE
- RX7700XT
- RX7800XT
- RX7900XT

## 内存

分为DDR4和DDR5。

DDR4频率：2133mhz、2400mhz、2666mhz、3200mhz、3600mhz、4000mhz、4266mhz。

颗粒选择海力士A-die。

时序：C28、C30、C32、C36。

频率：4800mhz、5200mhz、5600mhz、6000mhz、6400mhz、6800mhz、8000mhz。

推荐：

- DDR4 3200 C18 16G 8*2
- DDR4 3600 C18 16G 8*2
- DDR5 6000 C28 32G 16*2

## 硬盘

分为机械HDD和固态SSD(更好)，打游戏无所谓。

颗粒选择TLC。

接口：SATA、M.2(更好)。

通道：SATA、PCIe(更好，有3.0和4.0)。

协议：AHCI、NVMe(更好)。

走PCIe通道的固态基本都是NVMe协议。

### 推荐：

- 西部数据 SN580
- 铠侠 SD10

## 电源

根据CPU和显卡配置选择足够的瓦数就可以了。

> (CPU功耗 + 显卡功耗 + 100) * 1.2
>
> 列如：(cpu:100W + 显卡160W + 100) * 1.2 = 432W
>
> 这样购买额定功率为500W的电源即可

金牌、银牌比的是交流电转换直流电的效率。

功率：450W、500W、550W、650W、750W、850W、1000W。

推荐：

- 玄武 550V4 600w 铜牌
- 玄武 850K
- 海韵
- 振华
